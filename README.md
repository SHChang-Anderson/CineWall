# CineWall 🎬

**CineWall** is a lightweight poster-wall movie player built with **Python, PyQt5, and MPV**.  
It scans your local folders (or mounted **Google Drive / SMB**) and automatically fetches movie posters from [TMDb](https://www.themoviedb.org/).  
Click a poster, and the movie plays instantly using the MPV backend.  

## ✨ Features
- Scan specified folders for video files (`.mp4`, `.mkv`, `.avi`, `.mov`)
- Fetch posters automatically via **TMDb API**
- Display movies in a **poster-wall grid layout**
- Click a poster to play instantly
- Uses **MPV player** for robust format support

## 📦 Installation
Clone the repository and install dependencies:

```bash
git clone https://github.com/<your-username>/CineWall.git
cd CineWall
pip install -r requirements.txt
```
